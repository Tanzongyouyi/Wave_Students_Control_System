## 系统概述
本系统是一个功能全面的学生信息管理系统，支持学生、课程、教师、班级、选课、成绩和统计等核心功能，适用于学校教务部门使用。系统采用C语言开发，具有高效、稳定、安全等特点。
---
## 安装与启动
### 安装环境要求
- 操作系统：Windows/Linux/MacOS
- 编译器：GCC 或其他支持C99标准的编译器，发布版本使用TCC编译器
- 终端或命令行工具
### 编译与运行
```bash
gcc main.c -o Wave_Students_Control_System.exe
./student_management
```

## 用户登录
系统启动后，会显示登录界面。默认管理员账号如下：
| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin  | admin123| 管理员|
输入正确的用户名和密码后，即可进入主菜单。

## 学生管理
### 功能列表
- 添加学生
- 编辑学生信息
- 删除学生
- 查看学生信息
- 学生列表显示
- 搜索学生（按学号、姓名、班级等）
- 排序学生（按学号、姓名、GPA等）
### 操作示例
**添加学生**
1. 选择“学生管理” → “添加学生”
2. 输入学生信息（学号、姓名、性别、出生日期、班级等）
3. 系统提示添加成功

## 常见问题
### Q1: 忘记管理员密码怎么办？
A1: 请联系系统开发人员重置密码。
### Q2: 如何批量导入学生数据？
A2: 目前系统不支持批量导入，需手动添加。
### Q3: 数据丢失怎么办？
A3: 使用“系统管理”中的“数据恢复”功能，从备份文件恢复。
